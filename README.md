# Smart Study Planner

A full-stack study planner web app built with React, Spring Boot, and PostgreSQL. It helps users manage tasks, organize them by category, view tasks in a calendar, search tasks, and track overdue work.

## Features

- Add, edit, and delete tasks
- Mark tasks as done
- Built-in and custom categories
- Delete custom categories and move tasks to `OTHER`
- Search tasks by title
- Calendar view
- Clickable calendar dates for quick task creation
- Overdue task section
- Responsive task management interface
